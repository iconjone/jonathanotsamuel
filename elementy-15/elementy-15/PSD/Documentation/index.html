Template overview - top

Hi, firstly I would like to thank you for purchasing my template. In this section I`ll briefly introduce Elementy features. I've Included PSD file in "PSD" folder. Layers are named, grouped and slices are named, ready to be export in PNG or JPG file if you want to change graphic element. Examples hover effect it are layers of "hover". If you do not need, you can simply turn off it. So lets start.
